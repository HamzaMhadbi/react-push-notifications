# Environment Variables

| Name               | Name in production  | Description                | Default value | Mandatory |
| ------------------ | ------------------- | -------------------------- | ------------- | --------- |
| PORT               | `-`                 | The port of the web server | `-`           | `No`      |
| REACT_APP_BACK_URI | CLIENT_APP_BACK_URI | The URL of the back-end    | `-`           | `Yes`     |
